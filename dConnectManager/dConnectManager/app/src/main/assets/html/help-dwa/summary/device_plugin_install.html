デバイスプラグインのインストールについて

下記のリンクを押下することで、GooglePlayからデバイスプラグインをインストールすることができます。

アイコン プラグイン名 説明
HOSTデバイスプラグイン Androidスマートフォンの主機能へのアクセスが可能になります。
Hueデバイスプラグイン Light profileで、hueの色の変更やOn/Offができるようになります。
HeartRateデバイスプラグイン Health profileで、BLE対応の心拍計から心拍数を取得できるようになります。
Spheroデバイスプラグイン

Light profileで、Spheroの色の変更やOn/Offができるようになります。

また、Drive Controller profileで、Spheroの操作を行う事も可能になります。
IRKitデバイスプラグイン Remote Controller profileで、IRKitから赤外線の送受信ができるようになります。
Android Wearデバイスプラグイン Android Wearに対応した時計を操作するためのプラグインになります。
Hitoeデバイスプラグイン Hitoeデバイスと接続し、生体情報（心拍数、加速度、心電波形）を取得する機能を提供するプラグインになります。
Slack Message Hookデバイスプラグイン Slackにログインし、メッセージの投稿、取得、受信などの機能を提供するプラグインになります。。
AWS IoTデバイスプラグイン AWS IoT経由で遠隔にあるデバイスを操作します。
