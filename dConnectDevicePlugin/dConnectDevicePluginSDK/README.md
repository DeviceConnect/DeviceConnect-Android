# Device Connect プラグイン SDK
Device Connect プラグイン SDK は Device Connect プラグインを作成するためのSDKです。

## 提供する機能

Device Connect プラグイン SDK では以下の機能を提供します。

- Device Connect Managerとの連携
- サービス管理
- プロファイル管理
- イベント管理
