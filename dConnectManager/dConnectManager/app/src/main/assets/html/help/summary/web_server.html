Webサーバについて

Webサーバを起動して、SDカードにorg.deviceconnect.android.managerというフォルダを作成し、この中にあるファイルにアクセスできるようにします。

ONにすることで、 Web Serverが起動します。

org.deviceconnect.android.managerにあるファイルは、全てHTTP経由でアクセスできるようになりますので、ユーザに対して許可ダイアログを表示して確認を行います。
問題がなければ、はいを押下してください。

Android OS 6.0以降では、SDカードの使用にパーミッション許可が必要になります。 以下のようなダイアログが表示されますので、こちらも問題がなければ、許可を押下してください。
