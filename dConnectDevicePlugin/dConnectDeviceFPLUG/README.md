# F-PLUGプラグイン

このF-PLUGプラグインでは、F-PLUGに付随している電力・温度・湿度・照度センサーを操作するための機能を提供します。

## 開発環境
Android Studio 2.2.1以上

## ビルドマニュアル
- [F-PLUGビルドマニュアル](https://github.com/DeviceConnect/DeviceConnect-Android/wiki/F-PLUG-Build)

## サポートするプロファイル

* humidity
* illuminance
* light
* powerMeter
* setting
* temperature
