【AWS IoT Manager 管理】

クラウド上のAWS IoT ManagerへのSubscribe / Unsubscribe設定を行います。

また、メニューから「AWS IoT設定」「デバイス認証」「ログアウト」を実行出来ます。

画面には、クラウド上に設定されているAWS IoT Manageの一覧が表示されます。

スイッチのOn/Offに連動して、該当AWS IoT Managerへの応答・イベント受信TopicのSubscribe / Unsubscribe設定を行います。

OnになっているAWS IoT Managerに対して、ServiceDiscoveryを行い、デバイス一覧を取得します。

  • 「AWS Iotと同期」ボタン

    • 「AWS IoTと同期」ボタンを押下すると、クラウド上に設定されているAWS IoT Manageの一覧再取得を行います。

  • メニュー
    • 「AWS IoT 設定」

      • 【AWS IoT 設定】画面が表示されます。

      Device Connect Managerが起動されていない場合は、確認ダイアログが表示され、【AWS IoT 設定】画面は表示されません。

    • 「デバイス認証」

      • 【デバイス認証】画面が表示されます。

      Device Connect Managerが起動されていない場合は、確認ダイアログが表示され、【デバイス認証】画面は表示されません。

    • 「ログアウト」

      • AWS IoTからログアウトを行い、【AWS IoT ログイン画面】が表示されます。
