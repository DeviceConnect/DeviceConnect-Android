# Android Wear プラグイン

このAndroid Wearプラグインでは、Android Wearに対応したスマートウォッチを操作する機能を提供します。

## 開発環境
Android Studio 2.2.1以上

## ビルドマニュアル
- [Android Wearビルドマニュアル](https://github.com/DeviceConnect/DeviceConnect-Android/wiki/AndroidWear-Build)

## サポートするプロファイル

* canvas
* deviceOrientation
* keyEvent
* notification
* touch
* vibration
