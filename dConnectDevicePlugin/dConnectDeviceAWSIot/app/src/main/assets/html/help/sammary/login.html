【AWS Iot ログイン画面】

AWS IoTへログインするための情報を設定します。

  • アクセスキーID

    • AWS IAMにて取得したアクセスキーIDを設定します。

    ※使用するアクセスキーIDには、AWS IAMにて管理ポリシーをAWS IoTに限定するなどのセキュリティ対策を実施したアクセスキーIDを使用して下さい。

  • シークレットアクセスキー

    • アクセスキーIDを取得した際に生成された、シークレットアクセスキーを設定します。

  • リージョン

    • アアクセスするAWS IoTが所属するリージョンを設定します。

  • 「ログイン」ボタン

    • 「ログイン」ボタンを押下すると、設定した情報を元にAWS IoTへログインします。

    ログインが成功すると、「AWS IoT Manager 管理」画面が表示されます。

    ログイン情報が間違っているとエラー表示が出ます。

    エラー内容を元にログイン情報を確認して下さい。

既にAWS IoT へログインしている場合、「AWS IoT ログイン画面」は表示されません。
