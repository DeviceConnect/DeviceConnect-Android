# Hitoeプラグイン

このHitoeプラグインでは、hitoeトランスミッターから心拍数や心電図などの値を取得するための機能を提供します。

## 開発環境
Android Studio 2.2.1以上

## ビルドマニュアル
- [Hitoeビルドマニュアル](https://github.com/DeviceConnect/DeviceConnect-Android/wiki/Hitoe-Build)

## サポートするプロファイル

* battery
* deviceOrientation
* ecg
* health
* poseEstimation
* stressEstimation
* walkState
