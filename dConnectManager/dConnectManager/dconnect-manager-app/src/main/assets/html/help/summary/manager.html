Device Connect Managerとは

Device Connect Managerはスマートフォン上で仮想サーバとして動作し、デバイスを操作するためのWeb APIを提供します。

Device Connect Manager向けに作成されたデバイスプラグインをインストールすることで、Web APIから操作できるデバイスを増やすことができます。

OMA GotAPI Ver1.1に準拠しています。
