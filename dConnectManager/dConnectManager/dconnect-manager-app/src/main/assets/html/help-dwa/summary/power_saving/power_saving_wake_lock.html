WakeLockについて

Android OS はシステムでユーザー操作がないことを検出するとすぐに、状況に応じてスリープモードになるように設計されています。

WakeLockを有効にすることでユーザー操作がないことを検出してもスリープモードにならないようにします。
ただし、スリープモードになりませんので、バッテリー消費が増加します。

バックグラウンドや外部IPから操作されるなどのユーザー操作がない場合でも動作させる必要がある場合は、WakeLockを有効にしてください。

Android OS 6.0以降では、Dozeモードが実装されています。
DozeモードになるとWakeLockを有効にしていても、WakeLockは無視されスリープモードになりますので、ご注意ください。
Android OS 6.0以降でスリープモード中にも動作させたい場合には、Dozeモードを無効化してください。
