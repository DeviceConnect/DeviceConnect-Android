Dozeモードについて

Android OS 6.0以降から、Dozeモードが実装されています。
DozeモードはAndroidデバイスを使用していない時にアプリケーションの動作を止めて(スリープ時の)バッテリーの減りを抑えるという機能です。

Dozeモードを無効化することで、アプリケーションがDozeモードにならないようにします。
ただし、Dozeモードにならない為に、バッテリー消費は増加します。

外部IPから操作されるなどのスリープ時でも動作させる必要がある場合には、Dozeモードを無効化にしてください。

無効化したDozeモードを有効にする場合には、バッテリー使用量を最適化画面からDevice Connect Managerを削除します。
