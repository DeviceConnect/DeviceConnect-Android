【AWS IoT 設定】

AWS IoTの設定情報を表示します。

また、AWS IoT ManagerのSubscribe / Unsubscribe 設定、イベントをPublishする周期設定を行います。

  • Device Connect Manager 名称

    • Device Connect Manager 名称が表示されます。

    スイッチのOn/Offに連動して、AWS IoT Managerの要求受信TopicのSubscribe / Unsubscribe設定を行います。

    同時にクラウド上のAWS IoT ManagerのOnline/Offline状態更新を行います。

    これにより、自機のDevice Connect Managerをクラウド上に公開します。

  • 同期周期

    • イベントで送信する情報をPublishする周期を秒単位で設定します。

    初期設定は10秒間隔です。0を設定すると即時送信となります。

    ※短時間で大量にイベントが発生するプロファイルのイベントの長時間連続受信は、AWS IoT利用課金が高額になる恐れがありますのでご注意下さい。

  • AWS IoT 情報

    • 以下のAWS IoTの設定情報を表示します。

    • MQTT endpoint
    • Request Topic
    • Response Topic
    • Event Topic
