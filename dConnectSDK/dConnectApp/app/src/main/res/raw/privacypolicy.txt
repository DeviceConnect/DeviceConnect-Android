Device Web API Managerに関するアプリケーション・プライバシーポリシー

本アプリケーション（以下、アプリ）・プライバシーポリシーは、 株式会社GClue が提供するスマートフォン向けアプリ”Device Web API Manager”における個人情報とその取り扱いについて説明するものです。
本アプリ・プライバシーポリシーの内容をご確認・ご理解のうえで”Device Web API Manager”をご利用ください。 

アプリ本体の説明

【個人特定できる情報】
個人特定ができる情報のサーバへの送信はおこなっていません。

【個人特定ができない識別情報】
個人特定ができない識別情報のサーバへの送信はおこなっていません。

【 株式会社GClue の問い合わせ窓口】 
本アプリに関するお問い合わせ、ご相談は以下の窓口でお受けいたします。 

担当窓口
E-mail : dwm@gclue.jp 

【 本アプリ・プライバシーポリシーの変更について】 
本アプリのバージョンアップに伴って、送信される利用者情報、目的、送信先が変更される場合があります。変更内容などは、新バージョンのアプリに付随するアプリ・プライバシーポリシーをご参照ください。
